Adrian Delgado

Hola, soy Adrián Delgado, un apasionado desarrollador web con experiencia en la creación de sitios web atractivos y funcionales. Me especializo en HTML, CSS y JavaScript, y disfruto trabajando en proyectos que desafían mis habilidades y me permiten aprender cosas nuevas. Mi objetivo es crear experiencias digitales que sean tanto visualmente atractivas como fáciles de usar.

Galería de Proyectos

Luna Studio

Diseño de una marca moderna para un estudio creativo. Se desarrolló identidad visual, logotipo y web responsive.

EcoTrack App

Aplicación móvil para seguir el consumo energético doméstico. Incluye paneles de datos e integración con dispositivos IoT.

Café Nube

Página web y fotografía de producto para una cafetería artesanal. Se priorizó la estética cálida y acogedora.

Urban Path

Sistema interactivo de mapas para rutas urbanas sostenibles. Enfocado en movilidad ecológica y diseño intuitivo.
